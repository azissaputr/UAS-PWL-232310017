## Deskripsi Project
Project ini merupakan implementasi workflow otomasi menggunakan n8n sebagai bagian dari Ujian Akhir Semester (UAS) mata kuliah Pemrograman Web Lanjut.

## Teknologi yang Digunakan
- n8n – Platform otomasi workflow berbasis node
- JSON – Format konfigurasi workflow

## Struktur Project
```
UAS-PWL-232310017/
├── README.md
└── test project 1 - v3.json   # File workflow n8n
```

## Cara Import Workflow ke n8n
1. Buka aplikasi n8n
2. Klik menu Settings → Import Workflow
3. Pilih file test project 1 - v3.json
4. Klik Import
5. Workflow siap digunakan
